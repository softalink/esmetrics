EsMetrics is running ✓

This server speaks the VictoriaMetrics + Prometheus HTTP API. The vmui React UI is not yet embedded — for now, point your existing Grafana / vmui at this host.

The build script that downloads and embeds the upstream vmui bundle (Apache 2.0, vendored unchanged) is scheduled for a later sub-phase per the project plan.

Quick query

curl 'http://HOST:PORT/api/v1/promql?query=up'

Ingest a sample

curl -XPOST --data-binary @metrics.txt 'http://HOST:PORT/api/v1/import/prometheus'
